1. This Solution receives as input a folder, which contains impages
2. The solution takes the folder of images and creates the HTML to present them in a Carousel
3. Once complete, copy/deploy the contents of the "Markup" to your desired destination
4. The "images" folder, must be a sibling of the "index.html"

Notes
1. The input images will be sorted. then loaded into the Carousel, in the sorted order.");
	The easiest way to ensure you receive the desired order, is to number the images; 1-Castle, 2-Planets...
2. The carousel can be presented in an iFrame by using the following tag in the host:
	<iframe width="600px" height="350px" frameborder="0" scrolling="no" src="index.html"></iframe>
